# Dan - Personal Portfolio

A modern, responsive, and sleek personal portfolio for a System Engineer/DevOps Professional.

## Features
- **Dark Theme Aesthetics:** Beautiful dark mode with cyan accents.
- **Glassmorphism Design:** Semi-transparent cards with backdrop filters.
- **Smooth Animations:** Elements fade and slide in upon scrolling using Intersection Observer.
- **Fully Responsive:** Adapts flawlessly to desktop, tablet, and mobile devices.

## Structure
- `index.html`: The main HTML file containing semantic structure.
- `css/style.css`: Stylesheet with CSS variables and custom classes.
- `js/script.js`: Interactive logic for navigation, scroll tracking, and animations.

## How to Run Locally
1. Clone the repository.
2. Open `index.html` in your favorite web browser.
3. Alternatively, to avoid CORS issues and simulate a real server, run:
   ```bash
   python -m http.server 8000
   ```
   and navigate to `http://localhost:8000` in your browser.
